Learn about resources to get started here!


Here are some useful places to go:

(Don't worry, they'll all open in a new tab.)

  • roll20.net

Roll20 is a free website to connect to friends and play D&D together.
You can create a free account, join a game (or make your own!), invite your friends, and finally, you can start playing!
You can also browse the marketplace for easy access to rulebooks, pre-made maps, character tokens, tilesets, and even entire modules.
There is also the D&D Compendium, which is very useful for looking up spell effects, rules, classes, races, items, etc.


  • dndbeyond.com

D&DBeyond is another good free website.
This website lets you create characters at any time, while Roll20 only lets you create characters if you're already in a game.
D&DBeyond has its own marketplace and compendiums of classes, spells, and more (found in the Game Rules and Sources dropdowns at the top of the page).

...However, since you can't directly play through D&DBeyond, you would need to buy resources from both Roll20 and D&DBeyond if you wanted to use them in both.


Along with those popular websites, below are some more resources recommended to me:

  • 5e.tools is yet another great website with a large database of different things. This might be the best database here. Highly recommended.

  • Here is a graphical page shared among the community that helps define a lot of things. Also highly recommended.

  • Of course, you can also use the official website as well.
